# HW2
# СИСТЕМА ДЛЯ УПРАВЛЕНИЯ РЕЦЕПТАМИ

Проект для управления рецептами, масштабирования порций и генерации списка покупок. Создан в рамках ДЗ№2 по Технологиям программирования.

## Описание

Проект позволяет:
- Создавать ингредиенты с названием, количеством и единицей измерения
- Создавать рецепты и добавлять в них ингредиенты
- Масштабировать рецепты под любое количество порций
- Создавать диетические рецепты с указанием категории (веган, без глютена и т.д.)
- Формировать список покупок из нескольких рецептов с автоматической группировкой ингредиентов
- Объединять списки покупок

## Установка и запуск
### 1. Клонирование репозитория

```bash
git clone https://github.com/reypoluc/HW2
cd HW2
## 2. Установка зависимостей
pip install -r requirements.txt
## 3. Запуск тестов
pytest test_recipes.py -v
## 4. Демонстрация работы
python recipes.py
```
